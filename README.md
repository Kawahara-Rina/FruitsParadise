# FruitsParadise(フルーツパラダイス！)

<dl>
  <dt>概要</dt>
  <dd>木からつぎつぎと落ちてくるフルーツをキャッチする落ち物アクションゲーム。たくさんのフルーツをゲットして、ハイスコアを目指そう！フルーツパラダイス！！2年時の個人制作作品。学園祭にて展示。</dd>

  <dt>プラットフォーム：Windows</dt>
  <dt>使用言語：C#</dt>
  <dt>開発環境：Unity、Visual Studio</dt>
  <dt>開発期間：2023年6月～9月</dt>
</dl>
